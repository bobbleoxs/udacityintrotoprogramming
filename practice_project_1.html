1. Concepts

Tags -
Opening tag and closing tag - mark up language

Elements -
Opening tag and closing tag and the text in the middle are together known as element

Attributes -
links and images need attribute to indicate that is what is being brought in.

Inline v Block -
inline - just text but tag br is special as it acts as starting a new line but it is still inline tag

block - makes an invisible box wrapping around text

Tags that are inline include: br, em, img, span, strong; block includes: p, div and form.
